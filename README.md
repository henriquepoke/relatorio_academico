# relatorio_academico

Este projeto em Python realiza a leitura de uma planilha Excel contendo dados de alunos e automatiza a análise das notas utilizando a biblioteca OpenPyXL.
A aplicação processa informações como nome, curso, idade, nota final e data de matrícula, gerando relatórios separados de alunos aprovados e reprovados, além de exibir estatísticas importantes no terminal.

Funcionalidades

Leitura de dados a partir de um arquivo alunos.xlsx
Separação automática de alunos aprovados (nota ≥ 7) e reprovados
Geração de duas novas planilhas:
- alunos_aprovados.xlsx
- alunos_reprovados.xlsx

Cálculo da média geral da turma
Identificação do aluno com a maior nota

Exibição no terminal de:
Total de alunos
Quantidade de aprovados e reprovados
Nota média da turma
Nome do aluno com a maior nota
